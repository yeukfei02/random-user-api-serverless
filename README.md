# random-user-api-serverless

random-user-api-serverless

## Requirement:

- install yarn
- install node (v12+)
- install serverless

## Testing and run:

```
// run function
$ serverless invoke local --function main

// test api in local
$ yarn run dev

// deploy to serverless
$ yarn run deploy

// open serverless dashboard
$ yarn run dashboard

// use eslint and prettier to format code
$ yarn run lint

// run test case
$ yarn run test
```
